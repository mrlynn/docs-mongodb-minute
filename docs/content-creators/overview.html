âœï¸ Content Creator Guide Overview

This guide will help content creators get started with MongoDB Minute, from initial setup to creating and publishing your first episode.

Prerequisitesâ€‹

Before you begin, ensure you have:

  • Node.js 18+ installed
  • MongoDB Atlas account (or access to MongoDB database)
  • MongoDB.com email for admin access
  • Code editor (VS Code recommended)

Quick Startâ€‹

1. Access the Applicationâ€‹

Navigate to the MongoDB Minute application URL. If you're setting up locally, the development server typically runs on http://localhost:3001.

2. Login with Magic Linkâ€‹

  1. Click the Admin link in the navigation
  2. Enter your MongoDB.com email address
  3. Check your email for the magic link
  4. Click the link to authenticate (valid for 15 minutes)
  5. You'll be redirected to the admin dashboard
Email Restriction

Only MongoDB.com email addresses can access the admin interface. This ensures secure access to content management features.

3. Explore the Dashboardâ€‹

The admin dashboard provides:

  • Statistics Cards: Overview of episodes by status and workflow stage
  • Review Queue: Episodes pending technical review
  • Quick Actions: Create episodes, access review queue, generate with AI

Creating Your First Episodeâ€‹

Step 1: Navigate to Episode Creationâ€‹

From the admin dashboard, click "Create Manually" or use the "AI Generate Episode" button for AI-assisted creation.

Step 2: Fill Out Episode Informationâ€‹

Basic Metadataâ€‹

  • Episode Number: Sequential number (optional)
  • Title: Clear, descriptive title
  • Slug: URL-friendly identifier (auto-generated from title)
  • Category: Select from predefined categories:
    • Data Modeling
    • Indexing
    • Atlas
    • Vector & AI
    • Atlas Search
    • Aggregation
    • Security
    • Migration
    • New Features
  • Difficulty: Beginner, Intermediate, or Advanced
  • Status: Draft, Ready to Record, Recorded, or Published

60-Second Script Structureâ€‹

Each episode follows a structured format:

SectionTimeDescription
Hook0-5sOpening hook to grab attention
Problem/Context5-15sProblem statement or context
Tip/Solution15-45sCore solution or tip
Quick Win/Proof45-52sResult or benefit demonstration
CTA + Tease52-60sCall to action and next episode tease

Step 3: Add Additional Informationâ€‹

  • Visual Suggestions: Notes for video production
  • Video URL: Link to published video (when ready)
  • Social Media Links: Platform-specific URLs (YouTube, TikTok, LinkedIn, Instagram, X)

Step 4: Save Your Episodeâ€‹

Click "Save" to create the episode. The workflow system will automatically initialize, setting the episode to Draft stage.

Understanding the Workflowâ€‹

MongoDB Minute uses a 3-stage workflow for quality control:

  1. Draft: Initial creation and editing
  2. Tech Review: Technical accuracy review
  3. Approved: Ready for video recording

Learn more in the Workflow & Approval System guide.
